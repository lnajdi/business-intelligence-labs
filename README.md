# Labs du cours BI

Dépôt du matériel de labs du cours BI.

Les étudiants clonent ou mettent à jour ce dépôt, puis travaillent dans le dossier du lab en
cours. Les labs sont publiés progressivement pendant le cours.

## Labs disponibles

| Lab | Dossier | Sujet |
|---|---|---|
| Lab 1 | `labs/lab01_hop_duckdb/` | Ingestion Apache Hop, exploration DuckDB, qualité des données, premiers KPI |

## Utilisation du dépôt

```bash
git clone https://github.com/lnajdi/business-intelligence-labs.git
cd business-intelligence-labs
```

Pour le Lab 1 :

```bash
cd labs/lab01_hop_duckdb
```

Suivre ensuite :

1. `SETUP.md` à la racine du dépôt pour l'installation des logiciels.
2. `labs/lab01_hop_duckdb/README.md` pour le déroulé du lab.
3. `labs/lab01_hop_duckdb/lab1_consignes.md` pour le travail demandé.

## Organisation du travail

Le Lab 1 se fait **en séance**, il n'y a **aucun rendu à remettre**. Ce dépôt est du matériel
de cours en lecture seule : ne pas committer de bases générées, de captures d'écran ou de
travaux personnels.
